Projeto: Sistema de Controle de Facas
Descrição

A classe Faca representa um objeto do mundo real que possui:

Nível de afiação (0 a 10)

Presença de ponta (true/false)

O objetivo do projeto é aplicar Programação Orientada a Objetos utilizando:

Encapsulamento

Métodos de comportamento

Regras de negócio

Validação de estado

 Métodos Implementados
 afiar(int nivelAdicionado)

Aumenta o nível de afiação da faca.

Regras de negócio:

O valor deve ser positivo

O nível máximo permitido é 10

 quebrarPonta()

Remove a ponta da faca.

Regras de negócio:

Não permite quebrar se já estiver sem ponta

Conceitos Aplicados

Encapsulamento (atributos privados)

Métodos como comportamentos

Proteção do estado interno do objeto

Validação com if

 Como Executar

Instanciar o objeto Faca

Utilizar os métodos afiar() e quebrarPonta()

Testar com valores válidos e inválidos
